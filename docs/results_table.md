# Результаты обучения моделей для предсказания сепсиса

## Общая информация

**Датасет:** PhysioNet 2019 Sepsis Prediction Challenge  
**Общее количество образцов:** 20,336  
**Обучающая выборка:** 16,269 (80%)  
**Валидационная выборка:** 4,067 (20%)  
**Длина последовательности:** 48 часов  
**Количество признаков:** 40

## Результаты

| Модель | Параметры | AUROC | Качество |
|--------|-----------|-------|----------|
| **LSTM** | 219,265 | **0.9908** | Отлично ⭐⭐⭐⭐⭐ |
| **Transformer** | 798,465 | **0.9901** | Отлично ⭐⭐⭐⭐⭐ |
| **Real Mamba** | 480,513 | **0.9788** | Очень хорошо ⭐⭐⭐⭐ |
| **GRU-D** | 164,689 | **0.5000** | Не обучилась ⭐ |

## Анализ результатов

### Лучшие модели:

1. **LSTM** (AUROC = 0.9908)
   - Лучший результат среди всех моделей
   - Среднее количество параметров (219K)
   - Стабильное обучение

2. **Transformer** (AUROC = 0.9901)
   - Практически идентичен LSTM
   - Наибольшее количество параметров (798K)
   - Требует больше времени на обучение

3. **Real Mamba** (AUROC = 0.9788)
   - Хороший результат при меньших параметрах
   - Самая быстрая в обучении
   - Перспективная архитектура

### Проблемная модель:

**GRU-D** (AUROC = 0.5000)
- Не показала результатов лучше случайного угадывания
- Возможные причины:
  - Нестабильность градиентов
  - Неподходящий learning rate
  - Архитектурные ограничения для данной задачи

## Выводы

✅ **3 из 4 моделей** показали превосходные результаты (AUROC > 0.97)  
✅ **LSTM и Transformer** показали практически идентичные результаты  
✅ **Real Mamba** демонстрирует хороший баланс между точностью и эффективностью  
❌ **GRU-D** требует дополнительной настройки гиперпараметров

## Рекомендации

Для практического применения рекомендуется использовать:
1. **LSTM** - для максимальной точности
2. **Real Mamba** - для баланса точности и скорости
